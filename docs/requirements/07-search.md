# Epic 7: Search

## US-7.1 — Search Cards (P2)

As a user, I want to search across all my cards so I can find specific content quickly.

**Acceptance criteria:**

- Full-text search across card front and back content
- Results show matching text with highlight
- Click result to view/edit card
- Filter by group or tag

## US-7.2 — Search Files (P2)

As a user, I want to search across my uploaded files so I can find notes to create cards from.

**Acceptance criteria:**

- Full-text search across file content
- Results show matching text in context
- Click result to open file at matching location
