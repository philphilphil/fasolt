# Epic 10: Mobile & PWA

## US-10.1 — Responsive Design (P1)

As a user, I want to use the app on my phone so I can study during commutes and downtime.

**Acceptance criteria:**

- All views usable on mobile screen sizes
- Touch-friendly card flip and rating buttons
- Readable markdown rendering on small screens

_Note: P0 views should be built mobile-friendly from the start. This story ensures all P1 views also work on mobile._

## US-10.2 — PWA Install (P3)

As a user, I want to install the app on my phone's home screen so it feels like a native app.

**Acceptance criteria:**

- Service worker for offline card review (cached due cards)
- App manifest with icon and splash screen
- "Add to home screen" prompt
- Syncs when back online
