# Epic 8: Sharing & Collaboration

## US-8.1 — Share a Group as a Deck (P2)

As a user, I want to share a group of cards as a public deck so others can study my cards.

**Acceptance criteria:**

- Generate shareable link for a group
- Shared deck is read-only for recipients
- Recipients can import (copy) the deck into their own account
- Sharing can be revoked

## US-8.2 — Browse Shared Decks (P2)

As a user, I want to browse decks shared by others so I can find useful material.

**Acceptance criteria:**

- Public deck listing with title, description, card count, author
- Search and filter shared decks
- Preview cards before importing
- Import copies cards into my account

## US-8.3 — Import Shared Deck (P2)

As a user, I want to import a shared deck so I can study someone else's cards with my own schedule.

**Acceptance criteria:**

- One-click import from shared link or browse view
- Cards copied into my account (independent scheduling)
- Optionally assign to a group on import
- Attribution shown (original author)
