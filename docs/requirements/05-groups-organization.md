# Epic 5: Groups & Organization

## US-5.1 — Create Groups (P1)

As a user, I want to create groups so I can organize my cards by topic (e.g., "Work — Kubernetes", "Cooking", "Guitar Chords").

**Acceptance criteria:**

- Name and optional description
- Group list view
- Empty state encourages adding cards

## US-5.2 — Assign Cards to Groups (P1)

As a user, I want to add cards to one or more groups so I can organize my study.

**Acceptance criteria:**

- Assign during card creation or from card detail
- A card can belong to multiple groups
- Bulk-assign from card list
- Remove card from group without deleting the card

## US-5.3 — View Group Contents (P1)

As a user, I want to see all cards in a group so I can review what I'm studying for a topic.

**Acceptance criteria:**

- Card list with front text preview, next review date, ease factor
- Sort by next review, creation date, or alphabetical
- Show due count badge

## US-5.4 — Edit and Delete Groups (P1)

As a user, I want to rename or delete groups.

**Acceptance criteria:**

- Rename updates everywhere
- Delete group does not delete cards (just unlinks them)
- Confirmation before delete

## US-5.5 — Tags on Cards (P2)

As a user, I want to tag cards with keywords so I can filter and find them easily.

**Acceptance criteria:**

- Add multiple tags to a card
- Autocomplete from existing tags
- Filter card list by tag
- Tag management (rename, delete unused)
